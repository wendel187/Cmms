🚀 Como Usar

  1. Acesse a página Técnicos
  2. Clique no botão ✏️ Editar de qualquer técnico
  3. Modifique os campos desejados
  4. Clique em 💾 Salvar
  5. Veja a confirmação com toast

Campos Editáveis por Entidade:

  • Técnico: Nome, Email, Telefone, Especialidade, Setor, Status
  • Equipamento: Nome, Código, Setor, Criticidade, Status, Última Manutenção
  • Ordem de Serviço: Status, Prioridade, Descrição, Data Fechamento

🔧 Troubleshooting

Modal não abre?

  • Verifique se está importando de modal.js corretamente
  • Abra o console (F12) e procure por erros
  • Certifique-se que modal.js está no caminho correto

Formulário vazio?

  • Verifique se a API está respondendo com os dados
  • Procure por erros de rede no DevTools (Network tab)
  • Veja se os nomes dos campos correspondem aos do backend

PUT não funciona?

  • Verifique se o backend está rodando (http://localhost:8080)
  • Verifique a resposta no Network tab do DevTools
  • Procure por erros CORS ou de validação
